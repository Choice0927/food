# 美食地点收藏助手 - 需求文档（简化版）

## 1. 项目概述

一个轻量级的个人美食/地点收藏管理工具，帮助用户记录和管理日常生活中发现的美食店、景点等地点信息。

### 核心功能
- 添加收藏地点（地图定位、图文描述、标签分类）
- 管理收藏列表（查看、编辑、删除）
- 按位置筛选（左上角定位，类似美团外卖）
- 基础用户系统（注册、登录）

---

## 2. 用户角色

| 角色 | 说明 | 权限 |
|------|------|------|
| 注册用户 | 普通用户，登录后使用 | 完整的收藏管理功能 |
| 访客 | 未登录用户 | 仅可浏览公开信息（如有）|

---

## 3. 功能模块

### 3.1 用户认证模块

| 功能 | 说明 |
|------|------|
| 注册 | 邮箱/手机号 + 密码注册 |
| 登录 | 邮箱/手机号 + 密码登录 |
| 个人中心 | 修改头像、昵称、密码 |
| 退出登录 | 清除登录状态 |

### 3.2 首页/收藏列表（核心页面）

**页面布局**：
- 顶部：定位选择器（左上角，显示当前城市）+ 搜索框
- 中部：收藏地点卡片列表
- 底部：添加按钮（悬浮或固定在底部）

**功能点**：

| 功能 | 说明 |
|------|------|
| 定位筛选 | 左上角显示当前城市，点击可切换城市，筛选该城市的收藏 |
| 搜索 | 按地点名称、标签搜索收藏 |
| 卡片展示 | 封面图、名称、城市、标签、评分（可选）|
| 下拉刷新 | 支持下拉刷新列表 |
| 上拉加载 | 分页加载更多 |
| 点击进入详情 | 点击卡片进入地点详情页 |

### 3.3 添加/编辑地点

**入口**：首页点击"+"按钮进入添加页

**表单字段**：

| 字段 | 必填 | 说明 |
|------|------|------|
| 名称 | ✅ | 店名/地点名称（最多50字）|
| 城市 | ✅ | 自动定位或手动选择 |
| 地址 | ✅ | 详细地址，支持地图选点定位 |
| 经纬度 | ✅ | 地图选点时自动获取 |
| 描述 | ❌ | 个人备注、推荐菜等（最多300字）|
| 图片 | ❌ | 最多9张，支持拍照或相册选择 |
| 标签 | ❌ | 如"火锅""咖啡""日料"等，支持自定义 |
| 评分 | ❌ | 1-5星个人评分 |

**地图选点功能**：
- 点击地址输入框旁的"地图选点"按钮
- 打开地图，可搜索地点或手动拖动标记
- 确认后自动填充地址和经纬度

### 3.4 地点详情页

**展示内容**：

| 内容 | 说明 |
|------|------|
| 图片轮播 | 多图滑动查看，支持放大 |
| 基本信息 | 名称、地址、城市、标签 |
| 地图位置 | 小地图展示位置，点击可导航 |
| 个人备注 | 描述、评分、推荐菜等 |
| 操作按钮 | 编辑、删除、分享 |

**操作功能**：

| 操作 | 说明 |
|------|------|
| 编辑 | 跳转编辑页，修改地点信息 |
| 删除 | 确认后删除该收藏 |
| 分享 | 生成分享卡片或链接 |
| 导航 | 调用手机地图APP导航到该地点 |

### 3.5 个人中心

**页面布局**：
- 顶部：用户信息卡片（头像、昵称、简介）
- 中部：统计数据（收藏数、标签数等）
- 底部：功能菜单列表

**功能菜单**：

| 菜单项 | 说明 |
|--------|------|
| 我的收藏 | 快捷进入收藏列表 |
| 个人资料 | 修改头像、昵称、密码 |
| 数据管理 | 导出收藏、导入收藏（可选）|
| 关于我们 | 应用介绍、版本信息 |
| 退出登录 | 退出当前账号 |

## 4. 数据模型（简化版）

### 4.1 用户表（User）

| 字段 | 类型 | 说明 |
|------|------|------|
| _id | ObjectId | 用户ID |
| email | String | 邮箱（登录用）|
| phone | String | 手机号（可选）|
| password | String | 加密密码 |
| nickname | String | 昵称 |
| avatar | String | 头像URL |
| createdAt | Date | 注册时间 |
| updatedAt | Date | 更新时间 |

### 4.2 地点/收藏表（Place）

| 字段 | 类型 | 说明 |
|------|------|------|
| _id | ObjectId | 地点ID |
| user | ObjectId | 所属用户（外键）|
| name | String | 地点名称 |
| city | String | 城市 |
| address | String | 详细地址 |
| location | Object | { lat: Number, lng: Number } |
| description | String | 描述/备注 |
| images | Array | 图片URL数组 |
| tags | Array | 标签数组 |
| rating | Number | 评分（1-5）|
| createdAt | Date | 创建时间 |
| updatedAt | Date | 更新时间 |

## 5. 技术栈

| 层级 | 技术 | 说明 |
|------|------|------|
| 前端 | Vue 3 + Vite | 框架和构建工具 |
| 前端 | Vant 4 | 移动端 UI 组件库 |
| 前端 | Vue Router | 路由管理 |
| 前端 | Pinia | 状态管理 |
| 后端 | Node.js + Express | 服务端框架 |

| 后端 | JWT | 用户认证 |

## 8. 总结

这是一个**轻量级的个人美食/地点收藏工具**，核心功能是：

1. 🔐 简单的用户注册/登录
2. ➕ 添加地点（地图定位 + 图文描述 + 标签）
3. 📋 查看收藏列表（支持城市筛选、搜索）
4. ✏️ 编辑/删除收藏
5. 👤 个人中心管理

不涉及复杂的社交功能（点赞、评论、关注）、打卡系统、行程规划等功能，保持应用的简洁和专注。
